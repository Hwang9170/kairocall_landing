# KairoCall 랜딩 페이지 디자인 방향

## 선택된 디자인: 현대적 미니멀리즘 + 신뢰감 중심

### 핵심 철학
**"신뢰할 수 있는 기술, 따뜻한 인간관계"**

KairoCall은 시니어 세대를 위한 AI 서비스이기에, 단순히 기술적 우월성을 보여주는 것이 아니라 **신뢰감과 안정성**을 시각적으로 전달해야 합니다. 동시에 현대적이고 세련된 인상을 주어야 합니다.

### 디자인 원칙

1. **신뢰감 우선**: 깔끔한 라인, 일관된 간격, 안정적인 색상 팔레트
2. **가독성 극대화**: 시니어 사용자를 고려한 충분한 폰트 크기, 높은 명도 대비
3. **따뜻한 인간미**: 차갑지 않은 색상, 부드러운 모서리, 자연스러운 흐름
4. **기술의 현대성**: 정교한 타이포그래피, 미묘한 애니메이션, 세련된 레이아웃

### 색상 철학

| 색상 | 값 | 역할 | 의미 |
|------|-----|------|------|
| **주 색상** | `#2563EB` (파란색) | 신뢰, 안정성 | 기술, 신뢰, 안전성 |
| **보조 색상** | `#10B981` (초록색) | 성장, 활기 | 긍정적 변화, 건강 |
| **배경** | `#FFFFFF` / `#F9FAFB` | 명확성 | 깨끗함, 신뢰 |
| **텍스트** | `#1F2937` / `#374151` | 가독성 | 안정적, 명확 |
| **악센트** | `#F59E0B` (황색) | 강조 | 따뜻함, 주의 |

### 레이아웃 패러다임

- **비대칭적 구성**: 단순한 중앙 정렬을 피하고, 텍스트와 이미지의 배치를 다양하게
- **섹션 간 명확한 분리**: 색상 변화, 여백, 시각적 구분선으로 섹션 구분
- **계층적 정보 흐름**: 중요한 정보부터 점진적으로 전개

### 시그니처 요소

1. **그래디언트 악센트**: 파란색에서 초록색으로의 미묘한 그래디언트 (강조 요소에 사용)
2. **원형 아이콘 배경**: 부드러운 원형 배경으로 아이콘 감싸기 (따뜻함 표현)
3. **수평선 구분자**: 섹션 간 세련된 구분선 (SVG 또는 간단한 border)

### 상호작용 철학

- **부드러운 전환**: 모든 버튼, 링크에 0.2-0.3초의 부드러운 트랜지션
- **명확한 피드백**: 호버 상태에서 색상 변화, 약간의 상승 효과
- **스크롤 애니메이션**: 요소가 화면에 나타날 때 부드러운 페이드-인 + 슬라이드-업

### 애니메이션 가이드

- **진입 애니메이션**: 0.6-0.8초의 ease-out, 약간의 지연으로 순차적 진입
- **호버 효과**: 0.2초 ease, 약간의 스케일 업 (1.02-1.05배) 또는 색상 변화
- **로딩 상태**: 미묘한 펄스 효과 또는 스피너

### 타이포그래피 시스템

| 용도 | 폰트 | 크기 | 굵기 | 사용처 |
|------|------|------|------|--------|
| **제목** | Noto Sans KR | 48-72px | 700-900 | H1, 섹션 제목 |
| **부제목** | Noto Sans KR | 24-32px | 600-700 | H2, 카드 제목 |
| **본문** | Noto Sans KR | 16-18px | 400-500 | 설명, 단락 |
| **작은 텍스트** | Noto Sans KR | 12-14px | 400 | 라벨, 주석 |

**폰트 전략**: Noto Sans KR을 주로 사용하되, 제목에서는 굵기(weight) 변화로 위계를 만들기. 한글 가독성을 최우선으로.

## 개선 사항 요약

### 기존 대비 개선점

1. **이모티콘 제거**: 모든 이모티콘을 제거하고 아이콘 또는 텍스트로 대체
2. **섹션 확장**: 기존 5개 섹션에서 7-8개 섹션으로 확대
   - 고객 후기 섹션 추가
   - FAQ 섹션 추가
   - 가격 정보 섹션 추가 (선택사항)
3. **시각적 개선**:
   - 배경 이미지 또는 그래디언트 추가
   - 더 정교한 카드 디자인
   - 개선된 버튼 스타일
4. **상호작용 강화**:
   - 더 부드러운 애니메이션
   - 마이크로 인터랙션 추가
   - 폼 유효성 검사 개선
